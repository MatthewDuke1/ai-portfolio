# AI Portfolio

Portfolio site showcasing three full-stack applications built with AI-assisted engineering. Designed for employers evaluating AI engineering skills.

Live at [matthewduke1.github.io/ai-portfolio](https://matthewduke1.github.io/ai-portfolio/)

## Featured Applications

| App | Stack | Description |
|-----|-------|-------------|
| Property Management Portal | Next.js, AWS Amplify, GraphQL | Admin dashboard with tenants, guest passes, towing, reports |
| Guest Portal | React Native, Expo, AWS Amplify | 2-step digital parking pass for apartment visitors |
| Registration Wizard | React, MUI, Cognito, Stripe | 6-step onboarding with multi-step orchestration and rollback |

## Setup

Static HTML -- no build step required. Hosted via GitHub Pages from the `docs/` directory.

```bash
# Local preview
open docs/index.html
```
