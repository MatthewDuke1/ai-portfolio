Birthday Email Automator

Compose and send personalized birthday emails with gift search integration. Import contacts from Google Calendar, Apple Contacts, Outlook, or CSV. Configurable heads-up reminders so you never miss a birthday again.

Overview

A static web app that lets you manage birthday reminders and send personalized emails with gift suggestions. No backend required -- runs entirely in the browser with EmailJS for delivery and localStorage for persistence.

Key Features

  • Email Composer -- Write and send birthday emails directly from the app via EmailJS
  • ICS / vCard Import -- Upload .ics or .vcf files from Google Calendar, Apple Contacts, or Outlook
  • CSV Import -- Flexible column detection, supports multiple date formats (YYYY-MM-DD, MM/DD/YYYY, January 5 1990), downloadable template
  • Gift Search -- Integrated gift idea lookup for each contact
  • Heads-Up Reminders -- Configurable advance notifications so you never miss a birthday
  • Settings Tab -- Configure EmailJS credentials in the UI (no rebuild needed)
  • CSV Export -- Back up your friends list at any time
  • CORS Error Handling -- Clear diagnosis with "Open in mail app" fallback

Tech Stack

LayerTechnology
FrontendVanilla HTML / CSS / JavaScript (zero dependencies)
EmailEmailJS (public key, service ID, template ID)
StorageBrowser localStorage
ImportICS parser, vCard parser, CSV parser (all client-side)
HostingGitHub Pages or AWS S3

EmailJS Template Setup

Create a template with these variables: {{to_email}}, {{subject}}, {{message}}, {{to_name}} (optional). That's it.

Import Formats

  • Google Contacts: Export as vCard (.vcf)
  • Google Calendar: Export as .ics
  • Apple Contacts: File → Export → Export vCard
  • Outlook: File → Import/Export → Export to CSV
  • CSV: Header row with Name, Email, Birthday, Relationship, Notes

Entire application designed and built with AI-assisted engineering in a single session. ICS/vCard parsing, CSV column detection with multiple date format support, EmailJS integration with CORS error handling, gift search, and AWS deploy script -- all generated and iterated through AI pair programming.

Overview

Guest-facing application where tenants register vehicles and visitors obtain digital parking passes through a simple 2-step process. Built with React Native + Expo for cross-platform deployment.

Key Features

  • Guest Parking Passes -- 2-step issuance: enter access code, submit vehicle info
  • Email Confirmations -- Passes emailed to guests via Lambda/SES
  • Expiry Notifications -- EventBridge-triggered alerts when passes approach expiry
  • Car Autocomplete -- Make/model suggestions via NHTSA vPIC API
  • License Plate Validation -- Format enforcement across all forms
  • Real-time Subscriptions -- Live updates via AppSync

Data Models

ModelPurpose
UserTenant account with property association
TenantRegistered occupant with vehicles
GuestVisitor with active parking pass
CarVehicle record (make, model, plate)

QuikSlip Registration Wizard

6-step onboarding wizard for new property managers. Collects personal details, complex information, and parking configuration, then provisions user accounts via AWS Cognito and stores property data in DynamoDB. Includes email verification and Stripe payment redirect.

Overview

Multi-step registration form for apartment complex managers to onboard their properties. Provisions user accounts via AWS Cognito and stores property data in DynamoDB through AppSync.

Registration Flow

StepScreenDetails
1Personal InfoName, email (duplicate check), phone, password
2Complex InfoComplex name, address, unit count, contact info
3Parking ConfigTotal spots, tenant/guest/admin/handicap allocation
4ReviewConfirm all data. Submit creates User + Cognito + Tenant
5Verify EmailEnter Cognito confirmation code
6CompleteSuccess + Stripe payment redirect

Orchestration with Rollback

Step 4 performs multi-step orchestration: create User in AppSync, sign up in Cognito, update User with Cognito ID, create Tenant. If any step fails, all previous operations are rolled back automatically.

AI Contribution

AI-assisted development for form UX, responsive confirmation redesign, license plate validation, car autocomplete, security improvements (removing hardcoded Cognito IDs), and documentation.

npm install
