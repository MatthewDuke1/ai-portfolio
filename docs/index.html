Overview

Admin portal for property managers to oversee parking operations at apartment complexes. Backed by AWS Amplify (AppSync, Cognito, DynamoDB, Lambda, SES).

Key Features

  • Dashboard -- Stats cards, daily activity report with email delivery, potential vehicles to tow
  • Guest Passes -- List, search, tow request (6 reasons), delete, 24-hour auto-expiry with countdown
  • Tenants -- CRUD with CSV bulk import, search by name/plate/car, permanent records
  • Roster -- Staff management with role-based filtering and VOIP call button
  • Violations -- Tracking with license plate anomaly detection (blank, null, special chars)
  • Lots -- Current property display with pulsing active indicator
  • Notifications -- Persistent bell icon with mark-as-read, dismiss, clear (localStorage)

Overview

Guest-facing application where tenants register vehicles and visitors obtain digital parking passes through a simple 2-step process. Built with React Native + Expo for cross-platform deployment.

Key Features

  • Guest Parking Passes -- 2-step issuance: enter access code, submit vehicle info
  • Email Confirmations -- Passes emailed to guests via Lambda/SES
  • Expiry Notifications -- EventBridge-triggered alerts when passes approach expiry
  • Car Autocomplete -- Make/model suggestions via NHTSA vPIC API
  • License Plate Validation -- Format enforcement across all forms
  • Real-time Subscriptions -- Live updates via AppSync

Overview

Multi-step registration form for apartment complex managers to onboard their properties. Provisions user accounts via AWS Cognito and stores property data in DynamoDB through AppSync.

Registration Flow

StepScreenDetails
1Personal InfoName, email (duplicate check), phone, password
2Complex InfoComplex name, address, unit count, contact info
3Parking ConfigTotal spots, tenant/guest/admin/handicap allocation
4ReviewConfirm all data. Submit creates User + Cognito + Tenant
5Verify EmailEnter Cognito confirmation code
6CompleteSuccess + Stripe payment redirect

Orchestration with Rollback

Step 4 performs multi-step orchestration: create User in AppSync, sign up in Cognito, update User with Cognito ID, create Tenant. If any step fails, all previous operations are rolled back automatically.

AI-assisted development for form UX, responsive confirmation redesign, license plate validation, car autocomplete, security improvements (removing hardcoded Cognito IDs), and documentation.
